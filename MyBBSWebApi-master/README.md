# MyBBSWebApi(BBS论坛开发(后端))
 ## 1.三层架构,UI---BLL----DAL----Models
```C# 
UI(UI层)引入依赖 BLL  和 Models
BLL(业务逻辑层) 引入依赖 DAL(数据访问层)
DAL(数据访问层) 引入依赖 Models(视图模型层)
```
